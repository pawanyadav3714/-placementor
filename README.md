# 🌟 About PlaceMentor AI

**PlaceMentor AI** is a next-generation AI-powered Placement Preparation Platform designed to help students prepare for coding interviews, technical assessments, aptitude tests, resume screening, and company-specific placement drives.

Unlike traditional coding platforms, PlaceMentor AI combines multiple AI models, personalized learning, intelligent analytics, and interactive interview simulations into one comprehensive ecosystem.

# 🚀 Installation

Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/placement.git
```

Install Dependencies

```bash
npm install
```

Run Development Server

```bash
npm run dev
```

Build

```bash
npm run build
```
